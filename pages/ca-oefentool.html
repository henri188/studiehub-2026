Cache Adresanalyse
Offset bits = log₂(blokgrootte)
Index bits = log₂(#sets)
Tag bits = rest

#sets = cachegrootte / (assoc × blokgrootte)

AMAT = HitL1 + MissL1 × (HitL2 + MissL2 × DRAM)
x86-64 Registers & Conventies

Return: rax
Callee-saved: rbx, rbp, r12-r15

push: rsp -= 8; [rsp] = val
pop: val = [rsp]; rsp += 8
2-complement

Gray Code
Bin → Gray: g_i = b_i XOR b_{i+1}
(MSB onveranderd)

Gray → Bin: b_0 = g_0
b_i = b_{i-1} XOR g_i
2-bit Sprongvoorspeller Staten
00 = Strongly NT (voorspel NT)
01 = Weakly NT (voorspel NT)
10 = Weakly T (voorspel T)
11 = Strongly T (voorspel T)

T → verhoog; NT → verlaag (verzadigt)
Carry Lookahead
p_i = a_i XOR b_i (1Δ)
g_i = a_i AND b_i (1Δ)
c_{i+1} = g_i + p_i · c_i

c_1: 2Δ; c_2,c_3,c_4: ook 2Δ
s_i = p_i XOR c_i: +1Δ
Soorten Cachemissers

Conflict: mapping-botsing, niet grootte

Conflict = miss bij DM maar niet bij FA
Capaciteit = miss bij FA
